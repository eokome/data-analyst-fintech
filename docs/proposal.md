## Reflection

This posting is directly relevant to ISBA 4715 because the Junior Data Analyst role at EPCVIP — a lead generation company that helps financial services clients acquire customers through digital advertising — requires the same skills covered in this course: SQL, building dashboards, and automating data pipelines. The role calls for monitoring campaign performance, ensuring data accuracy, and creating reports that drive business decisions, which maps directly to what we've practiced using tools like dbt (a tool that cleans and transforms raw data into analysis-ready tables using SQL), Streamlit for dashboarding, and GitHub Actions for scheduling. I'm building a pipeline that pulls consumer complaint data from the Consumer Financial Protection Bureau's public API, transforms it in Snowflake, and surfaces insights about which financial products and markets have the highest consumer dissatisfaction — the kind of demand signal an analyst at a lead generation company would use to guide campaign targeting. This project would also transfer to similar roles like Marketing Data Analyst, Business Intelligence Analyst at a bank or fintech startup, or Reporting Analyst at any financial services company.
